# T‑Travel — совместные поездки (Android)

Приложение для планирования совместных поездок: создавайте поездки, приглашайте участников, распределяйте бюджет по категориям и фиксируйте траты — приложение помогает понимать, **кто сколько потратил**, **кому кто должен** и сколько ещё осталось в бюджете.

## Возможности (по экранам)

- **Авторизация**
  - Вход по телефону и паролю
  - Регистрация (имя/телефон/пароль)
- **Поездки**
  - Список поездок и приглашений (принять/отклонить)
  - Создание новой поездки: название, маршрут, даты начала/конца
- **Бюджет**
  - Создание категорий расходов (например: отели, питание и т.д.)
  - Распределение общего бюджета по категориям
  - Редактирование распределения/сохранение изменений
- **Участники**
  - Просмотр списка участников (с отметкой администратора)
  - Добавление участников в поездку
- **Траты и операции**
  - Просмотр всех операций по поездке
  - Карточки операций/трат по участникам и категориям
  - Подтверждение/ожидание подтверждения операций (по статусам на экране)
- **Долги**
  - «Мои долги» и «Мне должны»
- **Профиль и “Ещё”**
  - Профиль пользователя и редактирование данных
  - История / уведомления
  - Выход из аккаунта

## Технологии

- **UI**: Jetpack Compose (Material / Material3)
- **DI**: Hilt (+ KSP)
- **Сеть**: Retrofit + OkHttp (+ logging-interceptor), Gson
- **Хранение**: Room (+ KSP)
- **Изображения**: Coil
- **Firebase**: Cloud Messaging, Crashlytics

## Сборка и запуск

1. Откройте проект в **Android Studio**.
2. Убедитесь, что установлен **JDK 11** (проект собирается с `jvmTarget = 11`).
3. Синхронизируйте Gradle и запустите конфигурацию `app`.
